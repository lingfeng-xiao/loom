# Loom 页面人工验收清单

## 会话首页
- 打开会话首页，确认顶部不出现“加载工作区快照失败”。
- 确认标题栏、左侧栏、主会话区、右侧轨迹栏同时完整显示。
- 确认主会话区铺满标题栏下方到输入区上方的可用高度，没有“半屏悬空”。

## 输入区
- 聚焦输入框，确认只有一层外部焦点边框。
- 确认输入框内部没有额外紫色框线或浏览器默认描边。
- 确认错误文案仅出现在输入区底部状态位，不覆盖主输入区域。

## 消息流
- 输入一条短消息并发送。
- 确认用户消息、思考态、助手消息都在主消息流中顺序出现。
- 确认消息流默认吸底，切换会话后不会停在错误的中间滚动位置。

## 右侧轨迹栏
- 确认轨迹和上下文面板仍然可见。
- 确认主区高度修正后，右侧栏没有被挤压或截断。
